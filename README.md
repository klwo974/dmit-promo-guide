## 常见问题

**Q：DMIT 优惠码一定要选季付以上吗？**  
大部分力度强的码（7折、8折）要求季付或年付。月付也有码，但折扣普遍只有 9 折。如果预算允许，年付组合下来能省最多。

**Q：特价套餐（WEE、MALIBU 等）还能用优惠码吗？**  
部分特价套餐不支持叠加优惠码，购物车验证时会提示。不行的话就以套餐原价为准，本身已经是低价入场了。

**Q：香港 T1 那个 45 折码是真的吗，配置还有升级？**  
是 DMIT 官方发布的，适用于 HKG T1 年付套餐，折扣同时附带配置升级（CPU 核数增加、硬盘翻倍、内存增加 50% 以上）。不过这类活动套餐有库存限制，早看早买。

**Q：支持哪些付款方式？**  
支付宝、微信、PayPal、信用卡都行，国内用户操作没有障碍。

**Q：IP 被墙怎么办？**  
每 15 天可以免费申请更换一次，超出频率的情况收费 $5/次。具体规则在官网 TOS 的 IP Replacement Policy 章节查得到。

**Q：Pro 系列和 Eyeball 系列线路质量差多少？**  
主要差在去程。Pro 是电联直连 CN2，EB 是 CN2/9929 混合。晚高峰下，Pro 的稳定性优势明显，EB 在大多数时候也够用，联通移动用户反馈 EB 性价比其实更高。
